MY 3D GAME: MORBUS

Steam banner created in Photoshop

MORBUS

'Morbus' is the second project I have done at Ncn. The objective was to create a 3D game using the Unreal engine, create 3D models using 3D Studio Max and add texture using Photoshop. We had to implement spatial storytelling: a way of putting across a story to the player without telling the story directly.
